CLOCs_LQS Framework（融合主框架）¶

CLOCs_LQS 主框架将所有 Stage 串联起来，形成完整的 2D–3D 融合检测流程。

主融合框架，将 2D/3D 检测器、各 Stage 算子串联形成完整融合流程。

API 文档¶

CLOCs_LQS_Pipeline¶

Bases: Module

CLOCs_LQS 融合主框架，将 2D 与 3D 检测结果通过候选匹配和 MLP 融合，得到最终的 3D 检测输出。

  • iou2d_thres (float, default: 0.0 ) –

    float 2D IoU 的配对阈值，IoU 大于该值的 2D–3D 组合会被视为有效候选对参与融合。

  • iou3d_thres (float, default: 0.5 ) –

    float 3D IoU 的 NMS 阈值，IoU 大于该值的 3D 检测框会被认为重叠过高，在后处理中被抑制。

class CLOCs_LQS_Pipeline(nn.Module):
    """
    CLOCs_LQS 融合主框架，将 2D 与 3D 检测结果通过候选匹配和 MLP 融合，得到最终的 3D 检测输出。

    Args:
        detector_2d: Detector2DInterface
            外部注入的 2D 检测器模块，实现从图像到 2D 检测框与分数的推理接口。
        detector_3d: Detector3DInterface
            外部注入的 3D 检测器模块，实现从点云到 3D 检测框与分数的推理接口。
        iou2d_thres: float
            2D IoU 的配对阈值，IoU 大于该值的 2D–3D 组合会被视为有效候选对参与融合。
        iou3d_thres: float
            3D IoU 的 NMS 阈值，IoU 大于该值的 3D 检测框会被认为重叠过高，在后处理中被抑制。

    Returns:
        forward(image, pointcloud, calib) -> (keep_indices, fused_scores)
            keep_indices: (M,)
                经过 3D NMS 后保留下来的 3D 检测框索引，对应原始 3D 检测器输出中的位置。
            fused_scores: (N,)
                每个 3D 检测框的融合置信度分数，结合了 2D 与 3D 检测信息，用于排序与后处理。
    """


    def __init__(self,
                 detector_2d: Detector2DInterface,
                 detector_3d: Detector3DInterface,
                 iou2d_thres: float = 0.0,
                 iou3d_thres: float = 0.5):
        super().__init__()

        self.det2d = detector_2d
        self.det3d = detector_3d
        self.mlp = FusionMLP()

        self.iou2d_thres = iou2d_thres
        self.iou3d_thres = iou3d_thres

    def forward(self, image, pointcloud, calib):
        """
        执行完整的 CLOCs_LQS 融合流程，将 2D 与 3D 检测结果融合为最终 3D 检测框输出。

        Args:
            image:
                输入图像数据，由 2D 检测器自行决定格式。
            pointcloud:
                输入点云数据，由 3D 检测器自行决定格式。
            calib: dict
                标定参数字典，应至少包含以下键值：
                - R0: (3,3)
                    相机坐标系矫正矩阵，用于对齐相机外参。
                - Tr_velo2cam: (3,4)
                    LiDAR → Camera 的坐标变换矩阵。
                - P2: (3,4)
                    Camera → Image 的投影矩阵，用于将 3D 框投影至图像平面。

        Returns:
            keep_indices: (M,)
                NMS 后保留下来的 3D 检测框在原 3D 检测器输出中的索引。
            fused_scores: (N,)
                对每个 3D 检测框的最终融合置信度分数（融合 2D+3D 信息）。
        """
        boxes_2d, scores_2d = self.det2d(image)
        boxes_3d, scores_3d = self.det3d(pointcloud)

        K = boxes_2d.shape[0]
        N = boxes_3d.shape[0]
        proj_2d = torch.ops.clocs.project_3d_to_image(
            boxes_3d,
            calib["R0"],
            calib["Tr_velo2cam"],
            calib["P2"]
        )  # (N,4)

        iou2d_mat = torch.ops.clocs.compute_iou2d(boxes_2d, proj_2d)  # (K,N)

        pair_idx = torch.ops.clocs.pair_selector(iou2d_mat, self.iou2d_thres)  # (p,2)

        if pair_idx.shape[0] == 0:
            fused_scores = scores_3d.clone()
            keep = torch.ops.clocs.fused_nms(boxes_3d, fused_scores, self.iou3d_thres)
            return keep, fused_scores
        pair_feats = torch.ops.clocs.pair_feature_encoder(
            pair_idx,
            iou2d_mat,
            scores_2d,
            scores_3d,
            boxes_3d
        )                          # (4,1,p)
        pair_feats = pair_feats.unsqueeze(0)  # (1,4,1,p)
        pair_feats = pair_feats.float()
        pair_scores = self.mlp(pair_feats)    # (1,1,1,p)
        pair_scores = pair_scores.squeeze(0)  # (1,1,p)

        # 调用注册算子
        score_matrix = torch.ops.clocs.scatter_to_matrix(
            pair_scores, pair_idx, K, N
        )  # (1,K,N)

        fused_scores = torch.ops.clocs.score_aggregator(score_matrix)  # (N,)

        keep = torch.ops.clocs.fused_nms(
            boxes_3d, fused_scores, self.iou3d_thres
        )

        return keep, fused_scores

    执行完整的 CLOCs_LQS 融合流程，将 2D 与 3D 检测结果融合为最终 3D 检测框输出。

    Args:

                Camera → Image 的投影矩阵，用于将 3D 框投影至图像平面。

        calib["R0"],

        calib["P2"]

        boxes_3d, fused_scores, self.iou3d_thres
    )

    return keep, fused_scores
